✦ AI / Machine Learning

Jessie-1 — AI Gym Coach

A custom AI-powered fitness assistant built using a local large language model combined with a retrieval-based system (RAG). Instead of generic responses, the system uses a structured gym knowledge base covering science-based lifting, workout splits, nutrition, recovery, and calisthenics.

Key Features

  • Answers gym, workout & nutrition questions with context-aware responses
  • Retrieval-Augmented Generation (RAG) for domain-specific accuracy
  • Personalized, conversational tone — feels human, not robotic
  • Runs fully locally via Ollama — zero external API dependency
  • Interactive chat UI built with Streamlit

What I Built

01Designed & structured a custom fitness dataset
02Built a Chroma vector database for semantic search
03Integrated a local LLM via LangChain + Ollama
04Engineered prompts for human-like responses
05Developed the full interactive chat interface

Tech Stack

Python LangChain Ollama (LLM) Chroma DB Streamlit

Demonstrates how modern AI assistants are built by combining LLMs with domain-specific knowledge — highlights real-world AI engineering beyond basic usage.
